⚔ 历史战争 — 蓄势爆发型
角色 净角(78%)构成对立主体，老生(74%)统帅角色，武生(28%)武斗配角，形成"冲突对立"二元结构
主题 以「曹操·将军·同白」(T17)、「刘备·诸葛亮·张飞」(T2)、「周瑜·韩信」(T24) 军事智谋为核心
叙事 高潮位置最晚(38%处)，西皮为主(73%)，叙事密度前低后高，层层蓄力至决战
对立角色积累张力 → 驱动军事主题 → 后置叙事高潮形成"蓄势爆发"
💕 爱情婚姻 — 情感先行型
角色 旦角(61%)比例全类型最高，搭配小生，情感驱动型双主角结构
主题 以「情节·角色」(T20)、「关公·夫人·情感」(T19) 情感叙事为核心，主题分布均匀
叙事 高潮位置最早(22%处)，西皮主导(74%)，情感冲突早呈现，后段多为情感流动收尾
旦+小生情感结构 → 推动感情/关系主题 → 高潮前置的"情感先行"叙事
⚖ 冤案公案 — 积怨渐发型
角色 角色复杂度最高(avg 6.9)，丑角比例突出(68%)，官-民-反派三角互动模式
主题 「情节·戏考」(T20) + 「伍员·复仇·申冤」(T8) 冤情与申诉主题并重
叙事 二黄比例全类型最高(39%)，叙事前段低密度铺垫，后段逐渐上升，体现苦诉积累
多角色冤情博弈 → 层叠冤情主题 → 前慢后急的"积怨渐发"叙事
🏯 历史宫廷 — 权谋沉郁型
角色 老生主导权威，净角+太监形成权力层级关系网，人物地位分层鲜明
主题 「包拯·太监·宫廷」(T15)、「太监·孙权·荆州」(T21) 围绕权力秩序展开
叙事 二黄占比偏高(29%)，曲调沉郁，叙事密度起伏平缓，场次最长(11场)
权力分层角色结构 → 宫廷权谋主题 → 二黄沉郁基调的"权谋沉郁"叙事

三要素协同演化四条规律
三维度相互约束、协同演化的稳定规律

规律一 · 角色对立度 → 叙事高潮位置
净角+老生对立结构越强，叙事高潮越靠后（延迟爆发，蓄势逻辑）； 旦+小生情感结构则对应高潮前置，因情感冲突需早建立。

规律二 · 主题复杂度 → 角色网络规模
主题越层叠（冤案多层冤情、忠义多方博弈），角色数量越多(6.6–6.9)， 体现"主题的层叠需要更多角色承载"的结构对应关系。

规律三 · 唱腔情绪 → 主题色彩
西皮（明快激昂）→ 军事/忠义/爱情主题； 二黄（沉郁悲凉）→ 宫廷权谋/冤情主题。 唱腔选择是主题情绪基调的音乐化表达，两者形成稳定映射。

规律四 · 旦角密度 → 叙事情感化程度
旦角比例越高，叙事密度越均匀（无明显峰值），整体呈"情感流动"型； 净角主导则叙事起伏更大，体现行当性格对叙事节奏的直接塑造。

三要素互锁结构示意
三要素并非独立，而是形成相互约束的"稳定结构单元"
